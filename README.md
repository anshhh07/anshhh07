## ◈ Featured Projects

<details>
<summary><b>🤖 AI Learning Platform</b> — Production-Grade RAG-Powered Educational System</summary>

<br>

A production-grade, **RAG-powered adaptive learning platform** that replaces static course content with dynamically assembled, context-aware educational material. The system ingests heterogeneous knowledge sources — documents and YouTube videos — embeds them into ChromaDB, and retrieves the most semantically relevant content per learner query with significantly reduced hallucinations.

| Attribute | Detail |
|:---|:---|
| **Stack** | Python · LangChain · ChromaDB · Gemini API · Django · React · Tailwind CSS |
| **Scale** | Multi-document corpus + YouTube video ingestion · Concurrent learner sessions |
| **Performance** | Semantic retrieval via ChromaDB · Grounded answer generation via Gemini API |
| **Security** | API key isolation · Rate-limited endpoints · Input validation on ingestion |
| **Impact** | Transforms static learning resources into dynamic, personalised knowledge delivery |
| **Repository** | [anshhh07/AI-learning-Platform-using-RAG.](https://github.com/anshhh07/AI-learning-Platform-using-RAG.) |

**ScholarYT** — a custom sub-component built inside this platform — performs **automated YouTube video segmentation and semantic summarisation**, transforming video content into indexed, queryable knowledge chunks. The Django backend handles document ingestion and API routing while the React conversational frontend provides a seamless cross-device learning experience.

</details>

| Attribute | Detail |
|:---|:---|

| **Scale** | Multi-tool integration · Real-time calendar and messaging APIs |
| **Performance** | Autonomous task decomposition · Near-zero manual intervention |
| **Security** | Scoped agent permissions · API endpoint validation |
| **Impact** | Reduces event planning overhead from days of manual coordination to automated agent execution |
| **Repository** | [anshhh07/meda](https://github.com/anshhh07/meda) |

The FastAPI backend exposes agent orchestration endpoints, and the React frontend provides **real-time task status visibility** to event organisers. Agent communication, tool routing, and multi-step task chaining are managed entirely by the CrewAI framework — demonstrating production-level agentic AI design.

</details>

| Attribute | Detail |

| **Performance** | Deterministic grounded retrieval safety layer · Graceful degradation without API key |
| **Security** | Environment-variable key isolation · Input validation · Scope-restricted agent |
| **Impact** | Automates assessment selection — reduces recruiter decision time from hours to seconds |
| **Repository** | [anshhh07/shl-assessment-recommender](https://github.com/anshhh07/shl-assessment-recommender) |

Architecture pairs an **OpenAI-powered conversational layer** with a **deterministic retrieval fallback** — grounded in real catalog data regardless of generative layer availability. Deployed with `/health` and `/chat` endpoints, demonstrating full production deployment maturity.

</details>
